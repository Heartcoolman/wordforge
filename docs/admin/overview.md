# 管理后台功能

管理后台是独立入口（`admin.html`），使用独立的认证体系（`ADMIN_JWT_SECRET`）。

## 初始化流程

1. 首次访问时系统检测管理员是否已初始化（`GET /api/admin/auth/status`）
2. 未初始化 → 引导创建首个管理员账户（`POST /api/admin/auth/setup`）
3. 已初始化 → 管理员登录页（`POST /api/admin/auth/login`）

## 功能模块

### 用户管理

- 查看/搜索全部用户列表
- 封禁/解封用户
- 双模式密码重置（直接重置 / 密钥自助）

### 系统监控

- 系统健康状态（运行时长、版本、数据库大小）
- 数据库详情（磁盘大小、树数量）
- AMAS 引擎性能指标和监控事件

### 数据分析

- 用户参与度（总用户数、今日活跃、留存率）
- 学习效果（总单词数、总记录数、整体正确率）

### AMAS 配置

- 在线查看/编辑 AMAS 算法参数
- 查看算法指标快照
- 查看监控事件列表

### 系统设置

- 注册开关
- 维护模式
- 用户上限
- 默认每日学习词数

### 全局广播

向全体用户推送系统通知（标题 + 消息内容）。
